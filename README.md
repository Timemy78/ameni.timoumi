### 📄 Retrieval-Augmented Generation (RAG) for Document QA

📌 Conception d’un système de Question-Réponse basé sur l’architecture RAG, permettant d’extraire des informations pertinentes à partir de documents non structurés.

🧠 Pipeline :
1. Ingestion des documents (PDF, DOCX)
2. Chunking & embeddings
3. Stockage dans une base vectorielle
4. Retrieval des passages pertinents
5. Génération de réponse avec LLM

⚙️ Stack :
- LangChain
- FAISS
- HuggingFace / GROQ / Ollama
- Streamlit

📈 Use Cases :
- Analyse documentaire
- Support client automatisé
- Knowledge base intelligente

🔗 [Voir le repo] (https://github.com/Timemy78/rag-document-qa)
